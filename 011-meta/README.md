# HTML 011：網頁元數據標籤 (Meta)

## 內容
- **`<meta>` 標籤**：位於 `<head>` 內部，用來提供網頁的「元數據」（給瀏覽器、搜尋引擎、社群平台看的資訊），不會顯示在網頁主體中。
- **重要屬性**：
  - `charset="UTF-8"`：設定網頁編碼，防止中文亂碼。
  - `name="description"`：網頁的簡介，Google 搜尋結果看到的摘要就是它。
  - `name="keywords"`：網頁的關鍵字。
  - `name="viewport"`：控制移動端裝置的縮放比例，是做響應式網頁（RWD）的必備標籤。
