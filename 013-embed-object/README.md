# HTML 013：外部資源嵌入標籤 (embed & object)

## 內容
- **`<embed>` 標籤**：是一個單標籤（不需要寫閉合標籤），用來快速嵌入外部或本機的多媒體資源。這裡用來嵌入本機的 `meow.pdf`。
- **`<object>` 標籤**：功能比 `embed` 更強大，是一個雙標籤。最大的優點是支援「後備內容（Fallback）」——如果瀏覽器沒辦法載入該 PDF 檔案，就會改為顯示標籤內寫的替代文字與下載連結。

## 本地檔案路徑重點
- 必須把 `meow.pdf` 檔案移到跟 `index.html` 相同的資料夾內。
- 屬性路徑直接寫檔名（如 `src="meow.pdf"` 或 `data="meow.pdf"`），這叫做**相對路徑**。
- 因為瀏覽器安全性限制，檢視本機 PDF 建議使用 **Live Server** 擴充功能開啟網頁，畫面才能順利載入。
